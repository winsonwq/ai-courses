# UI Design Specification

本项目遵循以下视觉设计规范，AI 在生成或审核代码时必须严格遵守。

## 1. 颜色规范 (Brand Colors)
- **Primary**: `#3B82F6` (Blue-500)
- **Secondary**: `#10B981` (Emerald-500)
- **Danger**: `#EF4444` (Red-500)
- **Background**: `#F9FAFB` (Gray-50)

## 2. 间距规范 (Spacing)
采用 4px/8px 网格系统：
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px

## 3. 圆角规范 (Border Radius)
- `button`: 6px
- `card`: 12px
- `modal`: 16px

## 4. 按钮检查规则
- 所有的 `button` 必须包含 `transition` 效果。
- 所有的 `primary button` 文字颜色必须为白色。
